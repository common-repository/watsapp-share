== Installation ==

1. Download the plugin and extract the files
2. Upload "whatsapp-share.zip" to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Check the "WhatsApp share" Menu created by this plugins for setup.
